# DataNarasi - Project Context (2026-04-28)

## 🎯 Project Overview
**DataNarasi** is a full-stack web application that automates data analysis. Users upload CSV/Excel files, and the system performs automated data cleansing, statistical analysis, chart generation, and ultimately uses an AI provider to write a business narrative/insight in Bahasa Indonesia.

## 🏗️ Architecture & Tech Stack
- **Backend (Laravel 13):** Handles user authentication (Breeze), file uploads, database management (MySQL), and asynchronous background jobs (Queue).
- **Frontend (Vue 3 + Inertia.js + TailwindCSS):** Provides a reactive SPA interface for users to upload data and view reports.
- **Microservice (Python 3.11 + FastAPI):** Handles heavy data processing. It uses Pandas for data cleansing/analysis, Matplotlib/Seaborn for charts, and various OpenAI-compatible SDKs for AI narrative generation.

## 🤖 Multi-AI Fallback System
The core feature of this project is a resilient multi-AI fallback chain. If one AI provider fails (e.g., rate limit, timeout), the system automatically falls back to the next one in the chain.

**Current Fallback Order (Configurable via `AI_PROVIDER_ORDER` in `.env`):**
1. Gemini 3 Flash Preview
2. Kimi (Moonshot)
3. GLM (Zhipu AI)
4. NVIDIA NIM (Llama 3.1 8B Instruct)
5. MiniMax (M2.5)
6. Claude (Anthropic - Paid)

## 🔄 How the Processing Pipeline Works
1. User uploads a file in Laravel.
2. Laravel sends a POST request to Python FastAPI (`/process`).
3. Python FastAPI immediately returns `202 Accepted` and runs the pipeline in a `BackgroundTask` to avoid cURL timeouts.
4. Pipeline steps: Data Cleansing -> Statistical Analysis -> Chart Generation -> AI Narrative Generation.
5. Once done, Python sends an HTTP POST callback back to Laravel (`/api/v1/reports/{id}/callback`).
6. Laravel updates the report status, saves AI usage logs, and the UI updates automatically.

- **Provider Seeding:** Added NVIDIA NIM and MiniMax providers to the `AIProviderSeeder` to ensure accurate usage tracking and telemetry in the Laravel database.
